Pseudo Class:

Utilização de uma ação existente com um parâmetro. Por exemplo:

p:hover{color:red;}

P é a tag, hover é uma pseudo class e o color é o parâmetro desejado.

Pseudo Element ::

Agrega elementos periféricos nas tags

a::after{content:'TEXTO';}

Depois dos links, vai aparecer a palavra TEXTO

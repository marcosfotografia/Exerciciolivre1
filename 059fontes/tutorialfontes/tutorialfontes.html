COMO USAR FONTES SEM BAIXAR

FONTES Fontes-A Fontes-B Fontes-C Fontes-D Fontes-E

Para usar uma ou mais fonte sem baixar. Uso on-line

Acessar : Google fonts

Fazer a busca na caixa SEARCH FONTS

  1. Acessar : Google fonts
  2. Fazer a busca na caixa SEARCH FONTS
  3. Clicar no cartão da fonte
  4. Baixar a barra de rolagem e clicar no sina de + dentro do círculo so SELECT XYZ 123
  5. Clicar no @ IMPORT e copiar o conteúdo dentro do < STYLE >
  6. Colar o @import no CSS abaixo de @ charse t
  7. Abaixo do @ Import tem "CSS rules to specify families"
  8. Copiar "font-family: 'Handlee', cursive;" e colar na tag que vai usar essa fonte
  9. Exemplo: h1 {font-family: 'Handlee', cursive;} no CSS

    10. Para aparecer o cartão, clicar no ícone dos três quadrados com + no canto superior direito.

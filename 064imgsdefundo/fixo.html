TESTE DE IMAGEM DE FUNDO - TAMANHO width scroll

Para ajustar a imagem no fundo da página tem algumas opções de background-size:.

100% 100%; Deforma a imagem ao ajusta o tamanho da tela.

contain; Repete o fundo ou mostra o fundo com o background-color.

cover; resolve a situação. Mas somente para a viewport. Se tiver rolagem, a imagem rola junto.

body{ height: 100vh;
background-color: black;
background-image: url('imagens/wallpaper002.jpg');
background-position: center;
background-repeat: repeat;
background-size: cover; Ocupar toda a tela.Se tiver barra de rolagem,a imagem se move.
background-attachment: fixed; O fundo não se move

Para rolar é só por SCROLL ou deixar vazio que é o default.

}

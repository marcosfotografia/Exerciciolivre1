ESTUDOS DE IFRAME


Acessando o site para estudar e entender .

Basta colocar:

< iframe src="https://marcosfotografia.github.io/Exerciciolivre1/" frameborder="0"> </iframe>

Basta digitar iframe+enter que a tag já é criada

frameborder= define com ou sem da borda. É apenas binário. 0=sem e diferente de 0, é com borda.

No espaço entrea as tags, pode se colocar o link para o site abrir em uma aba, caso o navegador não aceite iframe.

Por padrão o iframe mede 300px x 150px

A configuração no CSS (ou style) predomina sobre a configuração na tag, in line.

No Css pode ter configuração, por exemplo:

{iframe:width:400px; height:600px; border: 1px solid red;}

O Parâmetro SCOLLING: auto, yes ou no, não está funcionando. Define ter ou não a barra de roalgem horizontl.

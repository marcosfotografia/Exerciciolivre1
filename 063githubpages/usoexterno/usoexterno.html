SITUAÇÃO

Precisando mexer no código mas em outro computador

Pré Requisito

Máquina com git e GitHub

SITUAÇÃO

Precisando mexer no código mas em outro computador

ABRINDO O GITHUB DESKTOP

  1. File > Options > Account
  2. Sign in do GitHub.com > Abre a janela para por User e Password > Aparece LET'S GET START com os repositórios
  3. Clica no Repositório e no botão CLONAR nome do Repositório que aparece abaixo
  4. Abre outra popup. Indicar para onde será clonado, indicar uma pasta.

CUIDADO!

Na tela de escolher o Repositório, a opção clonar GitHub está ok mas no clonar Git, dá erro.

Solução; File > Options > Git > SAVE

PROCESSO PARA SAIR DO PC PÚBLLICO

  1. Repositorio > Remover e marcar Also Move This Repositóry To the Recycle Bin > Botão Remove
  2. File > Options > Sign Out
  3. File > Optins > Git > Apagar o Usuário e E-Mail > SAVE
  4. Esvaziar a lixeira.

Criando página

Criar uma pasta no Windows Explores
Clicar com o direito e "Abrir com Code"
No Visual Studio Code, à esquerda, o ícone de duas folhas abre o navegador.
No explorer tem ícone de "+folha" que cria uma página, basta renomear para .html e +"Pasta" que cria pasta.
Na página que se abre, basta digitar "!" e dar enter que o código base é gerado. Trocar o lang="eng" por "pt-br".

Links

Basta digitar "A" e dar enter que já cria a linha de código.
O target= pode ser:
_self para mesma aba.
_blank para nova aba.
_top desfaz o frame e abre na janela toda.
_parent nome do frame abre dentro desse frame.

Envelopar

    Para fazer um texto virar link:
  1. Seleciona o texto
  2. Ctr + Shift + P
  3. Abre uma caixa. Digita ou seleciona;
  4. Wrap With Abbreviation
  5. A caixa limpa, digite: a, enter
  6. A tag de link HRF já é criada
  7. Clicar entre as aspas após HREF=""
  8. Ctrl + espaço, mostra o drop down de endereços internos

    9. Criar página pelo link

    1. Cria o link com a tag <a>
    2. Clic direito + ctrl sobre o link criado
    3. Aparece a msg de erro com a opção de Criar página

USANDO UMA FONTE ON LINE NO CSS EXTERNO

  1. Acessa o site que tem a fote a se copiar
  2. Clica no ícone do Fontes Ninja (verde) já instalado no browser
  3. Clica sobre a palava escrita com a fonte e copia o nome da fonte
  4. Abre o https://fonts.google.com/ e pesquisa a fonte pelo nome copiado
  5. Clica no ícon dos 3 quadrados com o sinal de mais, abre a janea lateral
  6. Clica no + ao lado do "Select alguma coisa +"
  7. Clica em Import e copia só o que tem entre as tags Styles "@import...
  8. Cola no CSS abaixo de @charset UTF-8
  9. Copia o CSS rules to specify families
  10. cola onde a fonte será usada. H1, H2 etc

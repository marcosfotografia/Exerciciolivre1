    Criando Link

  1. Tecla 'a' e ctrl, aparece a linha de href.
  2. Coloca o cursor entre as aspas e digita o endereço
  3. Depois das aspas põe o ' "target= _xxxxx"

      4. O target '_xxx' pode ser:

    • _self  > Abre o link na mesma aba
    • _blank  > Abre o link em nova aba
    • _top  > Abre o link na janela toda, desfaz o frame
    • Nome do frame  >_parent   ???

  4. 'rel' é opcional
